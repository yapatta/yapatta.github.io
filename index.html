I'm Yujiro Yahata, studying master of engineering in computer science at Keio University in Japan. I have keen eyes for implementing well-designed and optimized software systems which I then bring to reality through my code.

My interests are below:

  • Software Engineering : I'm interested in building robust, scalable, and efficient software systems which can be a good solution for business objectives. Currently, I'm focusing on the development between the backend and infrastructure. I'm happy when other colleagues use my system to make their work more efficient.
  • Cryptography : The process of putting mathematics into real-world systems is very interesting. Especially, I'm working on Secure Multi-party Computation (MPC) as a research topic, which enables parties to jointly compute a function over their inputs while keeping those inputs private. I see potential of MPC to be integrated into other systems like distributed machine learning.
  • Distributed System : I not only burn with a keen curiosity in designing applications with distributed systems such as Kubernetes, LXD, and Consul, but my interest extends to distributed algorithms such Raft and Paxos. I'm also trying to leverage modern cloud technologies like AWS and Google Cloud for more efficient system management.
